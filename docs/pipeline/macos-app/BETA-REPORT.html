⚑ Your hands-on check at Gate 3 (what headless testing can't cover)

Download the app and verify these interactive flows on your Mac:

  1. Capture → translate: press ⌃⌥Y, drag a box over on-screen text → confirm OCR + translation + the result popup + clipboard + that it auto-saved to the Notebook.
  2. Screen-Recording permission: first capture triggers the macOS prompt → grant → Quit & Reopen → confirm capture then works.
  3. Google sign-in: Preferences → Account → Sign in with Google → confirm the browser flow returns and the notebook syncs.
  4. Apple sign-in: same via Sign in with Apple (the new web flow) → confirm it completes.
  5. Cross-Mac sync (if you have a second Mac): sign in on both, add/edit/delete → confirm convergence + tombstones.
  6. Visual polish: spacing, contrast, light/dark, the 译 menu-bar glyph at Retina.
